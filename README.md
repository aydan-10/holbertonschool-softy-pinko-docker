# Softy-Pinko Docker Project

This project contains various tasks related to containerization using Docker.

## Task 0: Create Your First Docker Image
- Goal: Create a Docker image based on Ubuntu.
- Build command: `docker build -f task0/Dockerfile -t softy-pinko:task0 .`
- Run command: `docker run -it --rm --name softy-pinko-task0 softy-pinko:task0`
